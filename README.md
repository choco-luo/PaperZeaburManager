## 📦v1.0架設方式
1. 準備好需要使用的伺服器檔案(Bukkit、Spigot、Paper、Fabric...)

2. Zeabur 購買一台2vCPU | 8GB的專有主機

> 硬體配置會不夠把Minecraft Server開起來。
>
> (PZM完全針對使用Zeabur架設的情況，不維護其他平台的運作)
